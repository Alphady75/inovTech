Création d'un site vitrine d'une agence créative fictive (design graphique, photographie, développement web,
communication — au choix du groupe), destiné à convaincre de futurs clients de faire appel à ses services.
Pages à livrer
• Accueil (présentation de l'agence, services phares)
• Réalisations / Portfolio (grille de projets avec visuel et courte description)
• Équipe (présentation des membres fictifs de l'agence)
• Contact
Point d'attention layout
La grille de réalisations est idéale en CSS Grid (mise en page façon mur d'images) ; la page Équipe se prête
bien à des cartes en Flexbox qui s'alignent et se réorganisent au responsive.
Contraintes techniques (identiques pour les 3 projets)
• Sémantique HTML : header, nav, main, section, article, footer utilisés à bon escient
• Au moins 1 section en Flexbox et 1 section en Grid
• Responsive testé sur 3 breakpoints minimum (mobile ~375px, tablette ~768px, desktop ~1280px)
• Repo GitHub dédié à cette rotation, avec au moins 1 branche et 1 Pull Request par membre
• Au moins 1 conflit de merge rencontré et documenté dans le README
• Déploiement sur GitHub Pages
