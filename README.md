Création du site vitrine d'une agence créative fictive

1. Présentation du projet
Nom de l'agence (proposition: inovTech)

Slogan

Nous créons des expériences digitales qui marquent les esprits.

Contexte

Dans le cadre de ce projet, l'objectif est de concevoir un site vitrine moderne et responsive présentant une agence créative fictive spécialisée dans les services numériques.

Le site devra mettre en valeur les compétences de l'agence, présenter ses réalisations, son équipe et permettre aux visiteurs de prendre contact.

Le projet servira également à démontrer la maîtrise de HTML5, CSS3 (Flexbox et Grid), Git/GitHub ainsi que les bonnes pratiques du développement Front-End.

2. Objectifs

Le site doit permettre de :

Présenter l'agence
Valoriser ses services
Mettre en avant les projets réalisés
Présenter l'équipe
Faciliter la prise de contact
Offrir une expérience utilisateur moderne sur ordinateur, tablette et mobile
3. Public cible
